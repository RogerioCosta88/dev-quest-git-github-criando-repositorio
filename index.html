Criando Repositório

  1. Abrir o site do GitHub
  2. Logar na minha conta
  3. Clicar em New
  4. Dar um nome para o repositório
  5. Colocar uma descrição sobre do que se trata esse repositório
  6. Clicar em Create repository

    7. Obs.: é importante deixar o reposítorio como Public para que outras pessoas possam ver o repositório criado.
